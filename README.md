AquaGuard AI 🌿🤖

Intelligent Detection and Removal of Water Hyacinth

AquaGuard AI is a smart environmental project designed to detect, monitor, remove water hyacinth, and convert collected biomass into useful products such as biogas and biofertilizer.

Problem Statement

Water hyacinth is a fast-growing aquatic weed that:

- Blocks sunlight
- Reduces dissolved oxygen
- Harms fish and aquatic life
- Blocks irrigation and waterways
- Increases mosquito breeding

Proposed Solution

Our system integrates:

- AI Detection using CNN
- Drone Monitoring
- IoT Water Quality Sensors
- Smart Harvester Machine
- Biomass Reuse System

Features

- Real-time monitoring
- Automatic weed removal
- Smart alerts
- Sustainable biomass conversion
- Eco-friendly solution

Technologies Used

- ESP32
- OpenCV
- TensorFlow
